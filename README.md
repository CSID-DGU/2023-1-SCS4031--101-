# INTRODUCE
주제 : 색각 이상자의 교육 접근성을 개선하는 실시간 영상 보정 서비스

내용 : 웹캠으로 촬영중인 영상에 있어서, HSV색공간에 대하여 사용자의 상황에 따른 보정치를 적용하여, 실시간으로 OpenCV에 의한 색상 보정 기능을 제공.

### [주요 기능]

#### - 사용자 맞춤 HSV 색변환 및 테스트

#### - 테스트 결과 기반 근사 수치 선택 및 실시간 영상 변환

#### - PC 저장 영상 변환
